# EnvOps Frontend Console

This directory is the EnvOps frontend console workspace for public beta development and local integration.

## Tech Stack

- Vue 3
- Vite
- TypeScript

## Development Commands

```bash
pnpm install
pnpm dev
pnpm test:unit
pnpm typecheck
```
