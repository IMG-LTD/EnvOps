# EnvOps v0.0.2 Checklist

## Included in release

- `backend/`：Spring Boot 3.3 多模块后端基线，可通过 `cd backend && mvn -pl envops-boot spring-boot:run` 本地启动。
- `frontend/`：Vue 3 + Vite + TypeScript 控制台前端基线，默认通过 `pnpm dev` 对接本地后端。
- `README.md`：已更新为 `v0.0.2` 仓库首页，包含版本概览、Quick Start、文档入口、默认账号与限制说明。
- `docs/`：
  - `envops-v0.0.2-项目详细说明.md`
  - `envops-v0.0.2-开发技术说明.md`
  - `envops-v0.0.2-用户操作手册.md`
- `release/`：当前版本的 release notes 与 checklist。
- 仓库根目录中支撑本地开发与联调的必要配置。

## Current functional coverage

- 动态路由登录链路可本地体验。
- 资产中心支持主机浏览、凭据新增、分组与标签浏览。
- 检测中心支持检测任务与主机快照查询。
- 应用中心支持应用、版本、安装包、配置模板、脚本模板管理。
- `deploy/task` 支持列表、详情、主机明细、执行日志，以及 execute / retry / rollback / cancel。
- 任务中心支持聚合查询与跳转详情。

## Release gate

- `frontend/.env.prod` 中生产 API 地址仍为空，正式部署前必须明确前端访问后端的接入方式。
- 首页、系统管理、流量控制当前仍以展示页为主，发布说明中不应将其表述为完整生产能力。
- 前端尚未提供发布任务创建页与审批页，说明材料中应明确这是当前版本边界。
