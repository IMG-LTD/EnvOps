# EnvOps 0.0.7 Release Notes

## Summary

- Task Center 从统一看板推进为统一追踪入口。
- 保留 v0.0.6 统一列表和轻量详情抽屉。
- 新增统一任务完整追踪页和 `GET /api/task-center/tasks/{id}/tracking`。
- 完整追踪页提供基础信息、状态时间线、日志摘要、原模块日志/详情入口。

## Included scope

- 统一任务完整追踪页作为 Task Center 的二级查看入口。
- 用户先在统一任务列表中打开轻量详情抽屉快速判断任务，再按需进入完整追踪页。
- 完整追踪页仅展示基础信息、状态时间线、日志摘要和原模块日志/详情入口。
- 后端新增只读接口 `GET /api/task-center/tasks/{id}/tracking`，由 envops-task 读侧组装追踪视图。
- Deploy 历史任务允许降级展示。
- Database Connectivity 与 Traffic Action 只保证新任务完整追踪。

## Not included

- 不提供统一任务内重试或统一任务内取消入口。
- 不提供多任务编排能力。
- 不建设统一完整日志平台，完整追踪页仅提供日志摘要和原模块入口。
- 不补录 Database Connectivity 与 Traffic Action 全历史任务。
- 不引入新执行引擎抽象。
