# EnvOps v0.0.4 Release Notes

## Highlights

1. 后端 release gate 与本地启动入口已改为 reactor-safe 命令。
2. 前端 targeted unit gate 已有独立脚本，不再串跑无关 spec。
3. 完整 `pnpm --dir frontend test:unit` 已恢复绿色基线。
4. 系统账号密码已改为哈希存储与校验。
5. Deploy 创建契约已收敛到真实执行参数，不再对外承诺 `deployDir`。
6. Task Center 已明确降级为 deploy-only 队列视图。
7. Traffic 页面与接口已明确 skeleton / not-ready 边界，不再把占位能力包装成真实切流。
8. 资产中心数据库资源页已支持六类数据库的真实连通性检测，覆盖单检、已选批检、当前页批检和全部筛选结果批检。
9. 数据库连接凭据支持在资产记录中维护，响应不回显密码，编辑时密码留空表示沿用已保存密文；`warning` 仍保留为人工标记。

## Deferred scope

1. Traffic 真实外部网关接通仍不在 `v0.0.4` 范围内。
2. Task Center 仍不是跨域统一队列。
3. Deploy 大规模主机检索和更深执行器增强继续延期。
