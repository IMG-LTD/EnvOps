# EnvOps v0.0.1-beta

## Highlights
- 首个可对外说明的 EnvOps beta 版本。
- 后端与前端仓库结构已收敛为 `backend/` + `frontend/` 的公开发布基线。
- `deploy/task` 页面浏览链路已可本地体验，覆盖列表、详情、主机明细与执行日志。
- 登录密码规则与 i18n 字面量 `@` 相关阻塞问题已完成修复。

## Included scope
- Spring Boot 多模块后端基线，包含 `envops-boot` 启动入口与本地体验种子数据。
- Vue 3 控制台前端基线，包含动态路由登录流程与基础控制台壳层。
- 资产、应用、巡检、发布、任务编排首版控制台壳层。
- `deploy/task` 列表、详情、主机明细、执行日志，以及 execute / retry / rollback / cancel 相关执行链路。
- 面向公开 beta 的根 `README.md` 与本地启动说明。

## Known limitations
- 当前版本仍为 `0.0.1-beta`，范围聚焦可运行、可演示的首版基线。
- 生产环境前端 API 路由方式仍需确认，确认前不应将其视为正式部署方案。
- `envops-architecture-full.md` 与当前 `docs/` 不属于本次发布内容。

## Quick start
```bash
cd backend && mvn -pl envops-boot spring-boot:run
```

```bash
cd frontend && pnpm install && pnpm dev
```

体验账号：`envops-admin / EnvOps@123`
