# EnvOps 0.0.1-beta Checklist

## Included in beta
- `backend/`：Spring Boot 3.3 多模块后端基线，可通过 `cd backend && mvn -pl envops-boot spring-boot:run` 本地启动。
- `frontend/`：Vue 3 + Vite + TypeScript 控制台前端基线，已整理为公开 beta 使用的前端目录。
- `README.md`：对外 beta 首页，包含范围说明、本地启动方式、体验账号与已知限制。
- 仓库根目录中支撑本地开发与联调的必要配置。
- 与本次 beta 相关的已验证修复与整理，包括登录密码规则、i18n 字面量 `@` 阻塞修复，以及 `deploy/task` smoke 相关可体验链路。

## Explicitly excluded from beta content
- `envops-architecture-full.md`
- `docs/`

## Verification before release
- [x] backend 测试：`cd backend && mvn test`
- [x] frontend 单测：`cd frontend && pnpm test:unit`
- [x] frontend typecheck：`cd frontend && pnpm typecheck`
- [x] deploy/task smoke remains green
- [x] README commands and paths match the repo
- [x] no public docs still tell users to enter the old frontend directory

## Release gate
- 生产环境前端 API 路由方式仍需确认；在该方式确认前，不应把本 beta 作为非本地部署的最终说明基线。
